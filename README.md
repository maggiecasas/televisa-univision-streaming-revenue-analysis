# TelevisaUnivision Streaming Revenue Analysis

This project analyzes TelevisaUnivision’s revenue mix between advertising and subscription from 2023 to 2025, using publicly available quarterly financial reports.

## Objective

To understand how the company’s revenue is evolving as it transitions from traditional linear TV toward digital streaming through ViX.

## Data Source

* TelevisaUnivision Investor Relations (Quarterly Reports) : https://investors.televisaunivision.com/financial-reports/quarterly-reports/default.aspx

## Key Insight

Advertising remains the primary revenue driver, but shows signs of softening, while subscription revenue is more stable and increasingly important in supporting the company’s digital strategy.
